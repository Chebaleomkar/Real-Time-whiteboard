# Realtime Whiteboard Web App

## Overview

A collaborative and real-time whiteboard web application built using HTML, CSS, and JavaScript. The app leverages Socket.io for WebSocket communication, allowing multiple users to draw on the whiteboard simultaneously.

## Technologies Used

- **HTML**: Structuring the web content.
- **CSS**: Styling the user interface.
- **JavaScript**: Adding interactivity and functionality.
- **Socket.io**: Enabling real-time communication through WebSockets.

## Features

- **Real-Time Collaboration**: Multiple users can collaborate on the whiteboard in real-time, seeing each other's drawings instantly.

- **WebSocket Communication**: Utilizes Socket.io to establish WebSocket connections, enabling efficient and low-latency communication between clients and the server.

- **Responsive Design**: The whiteboard adapts to different screen sizes, ensuring usability across various devices.

- **Simple User Interface**: A clean and intuitive user interface for a straightforward drawing experience.
